Java中的自动装箱和拆箱

#为什么要有自动装箱和拆箱

  1. 将一个数值转换成一个类，可以使这个类有多种可以调用的方法。
  2. 基本类型的数据不是对象需要装箱才能和其他Object类型的子类共用同一个接口。

  1. int—>Integer

  3. short->Short
  4. char–>Character
  5. long–>Long
  6. double->Double
  7. float–>Float
  8. boolean->Boolean
